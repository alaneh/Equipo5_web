Atención alumno:

Puedes solicitar la asignación de un casillero, esta prestación tienen la duración del periodo escolar y, al término de éste, se debe desalojar para realizar las tareas de limpieza durante el periodo vacacional, para posteriormente, si lo deseas, realizar la solicitud de renovación de casillero.


Ya sea para la renovación o la solicitud de asignación de uno, tendrás que llenar el siguiente formulario, para el cual te solicitamos los siguientes documentos en formato PDF. Credencial del IPN vigente y horario de clases vigente.
